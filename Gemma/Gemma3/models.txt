Key Features
Multimodal Capabilities: Gemma 3 models (except the 1B version) can process both text and image inputs.
Function Calling: Gemma 3 supports native function calling, which allows it to interact with external tools and APIs.
Multilingual Support: These models support over 140 languages.
Context Window: These models feature a long context window of 128K tokens.

Access Methods
Google AI Studio: Test and prototype with these models directly in the Google AI Studio web interface.

Vertex AI: Enterprise users can access and tune these models through Vertex AI Model Garden on Google Cloud.
